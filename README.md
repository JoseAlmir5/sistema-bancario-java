# sistema-bancario-java

Projeto de estudo em Java – sistema bancário simples com POO

- Programação Orientada a Objetos
- Classes e métodos
- Entrada e saída de dados
- Estruturas de repetição
- Menu interativo

Funcionalidades:

- Saque
- Visualização de dados da conta
- Menu com opções

Tecnologias:
- Java
- Console
